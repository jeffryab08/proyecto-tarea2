# Procesador Automatizado de Imágenes y Generador de Reportes

Este proyecto escanea un directorio de imágenes de entrada para ajustar su escala, pasarlas a escala de grises, estampar una marca de agua corporativa de manera proporcional y catalogar de forma automática los metadatos originales en un reporte estructurado de Excel.

## Requisitos Previos

Asegúrate de tener instalado Python 3.10 o superior y los paquetes de dependencias listados en el proyecto.
